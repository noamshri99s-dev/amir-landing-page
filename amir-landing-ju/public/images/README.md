## ✅ הוראות הוספה:

1. **הכן את התמונות:**
   - המר את כל התמונות לפורמט PNG
   - ודא שהשמות מדויקים (case-sensitive)

2. **העלה את התמונות:**
   - העתק את כל קבצי ה-PNG לתיקייה `public/images/`
   - ודא שהשמות תואמים בדיוק לרשימה למעלה

3. **בדוק את התוצאה:**
   - רענן את הדף בדפדפן
   - התמונות יוצגו אוטומטית באתר

## 📐 המלצות טכניות:

- **פורמט:** PNG בלבד
- **גודל מינימלי:** 
  - תמונות Hero: 1920x1080px
  - תמונות מוצרים: 800x1000px
  - תמונת Proof: 800x1200px
- **איכות:** מומלץ להשתמש בתמונות באיכות גבוהה (72-150 DPI)
- **גודל קובץ:** נסה לשמור על קבצים קטנים יחסית (עד 500KB לכל תמונה) לביצועים טובים
