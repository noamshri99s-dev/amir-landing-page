# קטלוג תמונות לפי סוג מוצר

תיקיות:
- **rings** – טבעות
- **bracelets** – צמידים
- **earrings** – עגילים
- **necklaces** – שרשראות

## מוסכמת שמות קבצים

אפשר עם או בלי סיומת קובץ (png, jpg וכו') – הקטלוג יקבל את שני הסוגים.

- **אותו מוצר בכמה וריאציות (צבע/סגנון):**  
  `שם המוצר.וריאציה` או `שם המוצר.וריאציה.png`  
  דוגמאות:  
  - `אובל עם 2 אבני טיפות בצדדים.זהב` או `אובל עם 2 אבני טיפות בצדדים.זהב.png`  
  - `אובל עם 2 אבני טיפות בצדדים.זהב צהוב`  
  - `אובל עם 2 אבני טיפות בצדדים.כסף`  

- **מוצר בודד (ללא וריאציות):**  
  `שם המוצר` או `שם המוצר.png` (בלי נקודה לפני הוריאציה) – יוצג כמוצר אחד עם וריאציה "default".

אחרי הוספת תמונות, הרץ:
```bash
npm run catalog:build
```
כדי לעדכן את `public/data/catalog.json` ואת הקטלוג באתר.
